# Tamizh Arangam

The Living Museum of Tamil Civilization

An interactive 3D virtual museum web platform dedicated to Tamil history, heritage, and artifacts.

## Features

- **3D Virtual Museum** - Immersive exploration of ancient Tamil civilization
- **Artifact Gallery** - View artifacts with orbit controls
- **Interactive Timeline** - Scroll through Tamil history from 300 BCE to present
- **Virtual Tours** - Guided cinematic tours of Tamil heritage
- **8 Theme Galleries** - Sangam Age, Chola Dynasty, Pandya Dynasty, Temple Architecture, Tamil Literature, Maritime Trade, Weapons & Warfare, Daily Life & Culture

## Tech Stack

- React 18 + TypeScript
- Three.js (WebGL)
- Framer Motion
- Zustand
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build
